# Project title - FloodTrack

# Short project description
A web based application for flood and hazard reporting with GIS Heatmaps. The proposed FloodTrack system is responsible for planning, monitoring, and responding to disaster-related incidents such as flooding, clogged drainage systems, and road hazards.
