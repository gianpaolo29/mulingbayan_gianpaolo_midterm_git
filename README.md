# Midterm Examination Reflection
1. What is Git used for?
Git is used to track changes in files and manage different versions of a project. It also a popular software when it comes to changing file content.

2. What is the difference between Git and GitHub?
Git is a version control tool, while GitHub is an online platform used to store and collaborate on Git repositories.

3. Why is branching useful?
Branching is useful because it allows developers to work on features or fixes without affecting the main code. So when it comes to a big project you need a different branches so members can easily edit file without affecting others
